# Preparcial ADA - Práctica de Algoritmos

Este repositorio contiene las soluciones a los ejercicios prácticos de código y teoría.
Estas soluciones responden al archivo [final_exam_practice.md](final_exam_practice.md), que contiene el enunciado completo de la práctica.

## Ejecutar los scripts de Python (Ejercicios 1, 2 y 3)

Para ejecutar las pruebas de los ejercicios resueltos en Python, asegúrate de tener Python instalado y ejecuta los siguientes comandos en la terminal:

```bash
python exercise1_two_sum.py
python exercise2_fibonacci.py
python exercise3_largest_rectangle.py
```

## Ejecutar el script de TypeScript (Ejercicio 4)

Para ejecutar el archivo TypeScript, asegúrate de tener instaladas las dependencias de Node.js. 

```bash
npx tsx exercise4_longest_valid_parentheses.ts
```

## Resoluciones Teóricas (Ejercicios 5 y 6)

Las respuestas a los ejercicios de selección de estructuras de datos y algoritmos se encuentran en el archivo [soluciones_5_y_6.md](soluciones_5_y_6.md), y corresponden a los ejercicios planteados en [final_exam_practice.md](final_exam_practice.md).
